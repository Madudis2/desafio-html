O que é HTML?

Diferente do que muitos pensam o HTML* não é uma linguaguem de programação, ele é conhecido como linguagem de marcação e é usado para construir páginas de internet para as pessoas navegarem.

HTML: Linguagem de Marcação de HiperTexto

O que pode inserir?

Pode ser inserido imagens, vídeos, HiperTexto, paragráfos, textos, títulos e entre muitos outros.

O que são listas?

Listas no HTML são formas de organização, elas são dividias em duas.

Listas Ordenadas

São listas que são colocadas em ordem numérica, por exemplo:

  1. Banana
  2. Uva
  3. Maça
    4.

Listas não ordenadas

São listas que são feitas por marcação de pontos, por exemplo:

  • Pêra
  • Morango
  • Kiwi
    •
